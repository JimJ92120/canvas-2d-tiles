# canvas-2d-tiles

A small **engine** to render tile-based **2D canvas** with a **top-down** view.

View [demo](https://jimj92120.github.io/canvas-2d-tiles/).

---

# setup

|        |        |
| ------ | ------ |
| `node` | `>=20` |
| `npm`  | `>=10` |

### install

```sh
# clone repository
git clone <repository-name>

# install dependencies
npm install
```

---

# commands

```sh
# start development server
npm run start

# build
npm run build
```
